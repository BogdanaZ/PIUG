Tehnologiile folosite

Javascript

Toata aplicatia este scrisa in Javascript, atat cod cat si server.

HTML 5

Am folosit acest sistem pentru a-mi permite să fac modificarea aspectului paginilor web, precum și să fac ajustări la aspectul acestora.

CSS 3

Css3 mi-a permis să adaptez aplicația la diferite tipuri de dispozitive, cum ar fi ecrane mari, ecrane mici sau imprimante.

Angular 8

Am folosit cadrul Angular 8 pentru a crea această aplicație cu o singură pagină (single page app).

Express server cu nodeJs

Aplicația rulează pe un server nodejs, în mod special Expressjs. Am folosit serverul și pentru deploy pe paginile github.

Github

Guithub-ul este folosit in acest proiect atat ca si sistem de versionare cat si ca hosting.

Bootstrap

Am folosit ngBootstrap, ca Angular layer peste Bootstrap-ul normal pentru a mă ajuta să proiectez aplicația web mai rapid și mai ușor.

Scss

Scss este o extensie a CSS-ul care mi-a permis să folosesc variabile, nested rules, inline imports, functions, etc. Ma ajutat și să lucrez mai organizat sau să scurtez timpul de mențenantă.

Jasmine si Karma

Jasmine este un testing framework care ma ajutat să scriu unit tests pentru aplicația mea de Angular, iar Karma oferă instrumente utile care îmi facilitează apelarea testelor Jasmine în timp ce scriam cod.
